# ProxyLayoutTraining
Code examples from Responsive UI with Proxy Layouts training for Qt Academy.

## About this training
Training covers usage of `LayoutItemProxy` type in various scenarios - from creating controls which auto-adjust to available space in the container to full pages which change their layout after chaning size of the window. Learning to use the `LayoutItemProxy` will help you to create responsive UI for your QML application.

Be sure to check this free training out at [Qt Academy website](https://www.qt.io/academy)!

## Building the examples
Bulding the examples require at least Qt 6.6 installed and configured on your system with the `Quick` and `Core` packages, which are part of the standard installation. For instruction on setting up the Qt see the ["Get and Install Qt" page in Qt documentation](https://doc.qt.io/qt-6/get-and-install-qt.html).

Launch Qt Creator (or other IDE of your choice) and open `CMakeLists.txt` file placed in root directory. This will load all of the available examples. Each example is built as separate executable which you can run by changing the CMake target via the kit selection menu. You can also build each example independently by opening `CMakeLists.txt` file placed in example subdirectory.
